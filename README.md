# Быстрый старт

## 1. Установи зависимости
Открой папку программы в терминале и выполни:
```
npm install
```

## 2. Запусти программу
```
npm start
```
В консоли появится ссылка вида `http://127.0.0.1:4317/?t=...` — открой её в браузере.

## 3. Вставь ключ API
Справа в поле **«Ключ API»** вставь свой ключ Groq и нажми **«Сохранить»**.
Ключ сохранится автоматически — вводить его придётся только один раз.

## 4. Выбери рабочую папку
Нажми **«Выбрать…»** и укажи папку, в которой агент будет работать с файлами.

## 5. Опиши задачу и запусти
Напиши, что нужно сделать, и нажми:
- **«Запустить сейчас»** — агент начнёт работать сразу;
- **«В очередь»** — задача выполнится позже, по очереди.

## 6. Применение изменений
- Если агент показал список изменений — посмотри их и нажми **«Применить»** или **«Отменить»**.
- Если изменения применяются автоматически — резервная копия файлов всё равно сохраняется, можно безопасно откатить.

## Полезно знать
- Справа видно расход лимитов API (запросы и токены в минуту/сутки) — если лимит закончится, программа сама подождёт и продолжит.
- Программу можно закрыть и запустить позже — очередь задач не пропадёт.
